## 描述

当玩家在暂停菜单地图上右键点击帮派区域时触发该回调。

| 参数     | 说明                 |
| -------- | -------------------- |
| playerid | 点击帮派区域的玩家ID |
| zoneid   | 被点击的帮派区域ID   |

## 返回值

本回调不处理返回值。

该回调在游戏模式中总是优先触发。

## 示例

```c
public OnPlayerClickGangZone(playerid, zoneid)
{
    new string[128];
    format(string, sizeof(string), "您点击了帮派区域 %i", zoneid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}
```
