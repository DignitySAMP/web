## 描述

当玩家退出菜单时触发该回调函数。

| 参数名   | 说明             |
| -------- | ---------------- |
| playerid | 退出菜单的玩家ID |

## 返回值

该回调始终在游戏模式中优先触发

## 示例

```c
public OnPlayerExitedMenu(playerid)
{
    TogglePlayerControllable(playerid, true); // 当玩家退出菜单时解除冻结状态
    return 1;
}
```
