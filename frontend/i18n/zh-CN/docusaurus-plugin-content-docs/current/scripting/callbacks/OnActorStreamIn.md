## 描述

当角色被玩家客户端流加载时触发该回调。

| 参数        | 说明                   |
| ----------- | ---------------------- |
| actorid     | 已为玩家流加载的角色ID |
| forplayerid | 触发流加载操作的玩家ID |

## 返回值

在滤镜脚本中总是优先被调用。

## 示例

```c
public OnActorStreamIn(actorid, forplayerid)
{
    new string[48];
    format(string, sizeof(string), "角色 %d 已为您完成流加载。", actorid);
    SendClientMessage(forplayerid, 0xFFFFFFFF, string);
    return 1;
}
```
