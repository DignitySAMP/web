## 描述

当玩家离开帮派区域时触发该回调函数

| 参数名   | 说明                 |
| -------- | -------------------- |
| playerid | 离开帮派区域的玩家ID |
| zoneid   | 玩家离开的帮派区域ID |

## 返回值

该回调始终在游戏模式中优先触发

## 示例

```c
public OnPlayerLeaveGangZone(playerid, zoneid)
{
    new string[128];
    format(string, sizeof(string), "你正在离开帮派区域 %i", zoneid);
    SendClientMessage(playerid, 0xFFFFFFFF, string);
    return 1;
}
```
