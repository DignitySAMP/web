:::info

本页列出[玩家专属变量系统](../../tutorials/perplayervariablesystem)中使用的玩家变量类型（即 pvar 类型）。

:::

| ID  | 定义                            |
| --- | ------------------------------- |
| 0   | PLAYER_VARTYPE_NONE（无）       |
| 1   | PLAYER_VARTYPE_INT（整型）      |
| 2   | PLAYER_VARTYPE_STRING（字符串） |
| 3   | PLAYER_VARTYPE_FLOAT（浮点型）  |
