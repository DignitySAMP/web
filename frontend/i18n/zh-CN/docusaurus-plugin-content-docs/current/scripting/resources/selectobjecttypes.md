:::info

本页面列出[OnPlayerSelectObject](../callbacks/OnPlayerSelectObject)回调使用的所有物体选择类型。

:::

| 数值 | 宏定义                      | 描述                 |
| ---- | --------------------------- | -------------------- |
| 1    | SELECT_OBJECT_GLOBAL_OBJECT | 全局物体（地图物体） |
| 2    | SELECT_OBJECT_PLAYER_OBJECT | 玩家私有物体         |
