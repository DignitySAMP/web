:::info

[SetObjectMaterialText](../functions/SetObjectMaterialText) 函数包含两类参数 - 纹理文本对齐方式和纹理文本尺寸。本页列出文本对齐方式参数。

:::

| 值  | 定义                 |
| --- | -------------------- |
| 0   | 物体纹理文本左对齐   |
| 1   | 物体纹理文本居中对齐 |
| 2   | 物体纹理文本右对齐   |
