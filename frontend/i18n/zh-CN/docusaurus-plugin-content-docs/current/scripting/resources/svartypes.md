:::info

[GetSVarType](../functions/GetSVarType) 函数使用的服务器变量类型（简称 svar 类型）。

另请参阅：[服务器变量系统](../../tutorials/servervariablesystem)

:::

| ID  | 定义                            |
| --- | ------------------------------- |
| 0   | SERVER_VARTYPE_NONE（无）       |
| 1   | SERVER_VARTYPE_INT（整型）      |
| 2   | SERVER_VARTYPE_STRING（字符串） |
| 3   | SERVER_VARTYPE_FLOAT（浮点型）  |
