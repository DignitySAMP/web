:::info

本页面列出了所有可通过[SetPlayerMapIcon](../functions/SetPlayerMapIcon)函数使用的地图图标样式。

:::

| 数值 | 常量                      | 含检查点标记 | 雷达地图范围             |
| ---- | ------------------------- | ------------ | ------------------------ |
| 0    | MAPICON_LOCAL             | 否           | 仅限近距离               |
| 1    | MAPICON_GLOBAL            | 否           | 在范围内时显示于雷达边缘 |
| 2    | MAPICON_LOCAL_CHECKPOINT  | 是           | 仅限近距离               |
| 3    | MAPICON_GLOBAL_CHECKPOINT | 是           | 在范围内时显示于雷达边缘 |
