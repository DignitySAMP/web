## 如何获取室内 ID

在室内输入指令：

```c
/interior
```

系统将显示当前室内宇宙 ID。使用[SetPlayerPos](../functions/SetPlayerPos)前，请先设置[SetPlayerInterior](../functions/SetPlayerInterior)和[LinkVehicleToInterior](../functions/LinkVehicleToInterior)。
