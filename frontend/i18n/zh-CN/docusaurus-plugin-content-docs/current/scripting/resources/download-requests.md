---
title: 下载请求类型
sidebar_label: 下载请求类型
description: 下载请求类型定义（用于OnPlayerRequestDownload回调）
---

:::note

这些定义适用于[OnPlayerRequestDownload](../callbacks/OnPlayerRequestDownload)回调函数。

:::

| ID  | 定义             |
| --- | ---------------- |
| -1  | 未知下载请求     |
| 0   | 空下载请求       |
| 1   | 模型文件下载请求 |
| 2   | 纹理文件下载请求 |
