## 描述

检查指定菜单是否处于禁用状态。

| 参数        | 说明              |
| ----------- | ----------------- |
| Menu:menuid | 需要检查的菜单 ID |

## 返回值

**true** - 菜单已被禁用  
**false** - 菜单未被禁用

## 相关函数

- [IsValidMenu](IsValidMenu): 验证菜单 ID 有效性
- [IsMenuRowDisabled](IsMenuRowDisabled): 检查菜单行是否禁用
