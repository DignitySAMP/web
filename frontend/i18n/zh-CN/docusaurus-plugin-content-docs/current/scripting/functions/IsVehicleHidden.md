## 描述

检测指定车辆是否处于隐藏状态。

## 参数说明

| 参数名    | 说明            |
| --------- | --------------- |
| vehicleid | 要检测的车辆 ID |

## 返回值

**true** - 车辆已隐藏

**false** - 车辆可见

## 示例代码

```c
if (IsVehicleHidden(vehicleid))
{
    // 车辆已隐藏
}
else
{
    // 车辆可见
}
```
