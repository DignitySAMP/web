## 描述

设置角色的生命值。

| 参数名       | 描述                         |
| ------------ | ---------------------------- |
| actorid      | 要设置生命值的角色 ID        |
| Float:health | 要设置的生命值数值（浮点数） |

## 返回值

**true** - 成功

**false** - 失败（指定的角色不存在）

## 示例代码

```c
new gMyActor;

public OnGameModeInit()
{
    gMyActor = CreateActor(179, 316.1, -134.0, 999.6, 90.0); // 在武器店扮演销售员的角色

    SetActorHealth(gMyActor, 100.0);
    return 1;
}
```
