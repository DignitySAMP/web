## 描述

获取连接到车辆的驾驶室 ID

| 参数名    | 说明                  |
| --------- | --------------------- |
| vehicleid | 要获取驾驶室的车辆 ID |

## 返回值

驾驶室对应的车辆 ID，若无连接则返回 **0**

## 示例

```c
new cabId = GetVehicleCab(vehicleid);
```
