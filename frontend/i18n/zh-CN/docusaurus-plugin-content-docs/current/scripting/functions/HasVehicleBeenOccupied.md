## 描述

检测指定车辆是否曾被玩家占用过。

## 参数

| 参数名    | 说明               |
| --------- | ------------------ |
| vehicleid | 目标车辆的 ID 编号 |

## 返回值

**true** - 车辆存在被占用记录

**false** - 车辆从未被玩家占用

## 示例

```c
if (HasVehicleBeenOccupied(vehicleid))
{
    printf("车辆 %d 存在玩家占用记录", vehicleid);
    // 执行相关逻辑（如重置车辆状态）
}
else
{
    // 车辆保持初始生成状态
    RepairVehicle(vehicleid);
}
```
