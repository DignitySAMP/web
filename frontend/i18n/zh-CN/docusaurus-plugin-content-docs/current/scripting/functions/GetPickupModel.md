## 描述

获取拾取物的模型 ID。

| 名称     | 描述                           |
| -------- | ------------------------------ |
| pickupid | 需要获取模型 ID 的拾取物标识符 |

## 返回值

返回拾取物的模型 ID。

## 示例代码

```c
new g_Pickup;

public OnGameModeInit()
{
    g_Pickup = CreatePickup(1239, 1, 1686.6160, 1455.4277, 10.7705, -1);

    new model = GetPickupModel(g_Pickup);
    // model = 1239
    return 1;
}
```
