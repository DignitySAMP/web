## 描述

控制车辆车门的开关状态。

| 参数名          | 描述                                    |
| --------------- | --------------------------------------- |
| vehicleid       | 要设置车门状态的车辆 ID                 |
| bool:frontLeft  | 驾驶座车门状态（1 开启/0 关闭）         |
| bool:frontRight | 副驾驶车门状态（1 开启/0 关闭）         |
| bool:rearLeft   | 左后车门状态（若存在）（1 开启/0 关闭） |
| bool:rearRight  | 右后车门状态（若存在）（1 开启/0 关闭） |

## 返回值

**true** - 函数执行成功

**false** - 函数执行失败（指定的车辆不存在）
