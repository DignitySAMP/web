## 描述

销毁帮派区域。

| 参数名 | 说明            |
| ------ | --------------- |
| zoneid | 要销毁的区域 ID |

## 返回值

1：函数执行成功。

0：函数执行失败。指定的帮派区域 ID 不存在。

## 示例

```c
new gangZone;

public OnGameModeInit()
{
    gangZone = GangZoneCreate(1248.011, 2072.804, 1439.348, 2204.319);
    return 1;
}

public OnGameModeExit()
{
    GangZoneDestroy(gangZone);
    return 1;
}
```
