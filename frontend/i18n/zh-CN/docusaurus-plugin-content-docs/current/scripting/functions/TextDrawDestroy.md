## 描述

销毁先前创建的文本绘图。

| 名称        | 说明                                          |
| ----------- | --------------------------------------------- |
| Text:textid | 需要销毁的文本绘图 ID，由 TextDrawCreate 返回 |

## 返回值

该函数不返回任何特定值。

## 示例

```c
new Text:gMyTextdraw;

public OnGameModeInit()
{
    gMyTextdraw = TextDrawCreate(240.0, 580.0, "欢迎来到我的OPEN.MP服务器");
    return 1;
}

public OnGameModeExit()
{
    TextDrawDestroy(gMyTextdraw);
    return 1;
}
```
