## 描述

获取与当前车辆连接的拖车单元 ID。

| 参数名    | 说明                          |
| --------- | ----------------------------- |
| vehicleid | 需要查询拖车连接的目标车辆 ID |

## 返回值

成功返回拖车的车辆 ID 编号，若未连接拖车则返回**0**

## 示例

```c
new
	trailerId = GetVehicleTrailer(vehicleid);
DetachTrailerFromVehicle(trailerId);
```

## 注意事项

:::warning

该函数不适用于火车类轨道车辆

:::
