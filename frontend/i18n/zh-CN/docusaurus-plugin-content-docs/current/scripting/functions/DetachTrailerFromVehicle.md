## 描述

断开车辆与其拖车之间的连接（如果存在）。

| 名称      | 描述          |
| --------- | ------------- |
| vehicleid | 牵引车辆的 ID |

## 返回值

该函数不返回特定值。

## 示例

```c
DetachTrailerFromVehicle(vehicleid);
```
