## 描述

通过函数名获取公共函数的唯一标识符(ID)。

| 名称         | 说明                   |
| ------------ | ---------------------- |
| const name[] | 要查询的公共函数名称。 |

## 返回值

- 成功时返回函数 ID（ID 从**0**开始）
- 函数不存在时返回 **-1**

## 示例

```c
public OnFilterScriptInit()
{
    printf("OnFilterScriptInit的ID: %d", funcidx("OnFilterScriptInit"));
    return 1;
}
```
