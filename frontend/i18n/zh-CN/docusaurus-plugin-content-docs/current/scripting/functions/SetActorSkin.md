## 描述

设置角色的皮肤。

| 参数名  | 描述            |
| ------- | --------------- |
| actorid | 要设置的角色 ID |
| skin    | 要设置的皮肤 ID |

## 返回值

**true** - 成功

**false** - 失败（指定的角色不存在或无效）

## 示例代码

```c
new gMyActor;

public OnGameModeInit()
{
    gMyActor = CreateActor(179, 1153.9640, -1772.3915, 16.5920, 0.0000);
    SetActorSkin(gMyActor, 270); // 将角色皮肤从179更改为270
    return 1;
}
```
