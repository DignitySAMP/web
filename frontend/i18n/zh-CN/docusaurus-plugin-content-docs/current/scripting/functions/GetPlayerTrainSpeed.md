## 描述

获取玩家当前驾驶的火车的行驶速度

## 参数说明

| 名称     | 说明      |
| -------- | --------- |
| playerid | 玩家的 ID |

## 示例代码

```c
new Float:speed = GetPlayerTrainSpeed(playerid);
SendClientMessage(playerid, 0xFFFF00FF, "您的火车当前速度：%f", speed);
```
