## 描述

获取指定玩家帮派区域的当前闪烁颜色。

| 参数名   | 说明        |
| -------- | ----------- |
| playerid | 目标玩家 ID |
| zoneid   | 帮派区域 ID |

## 返回值

返回玩家帮派区域的 RGBA 闪烁颜色值。

**0:** 函数执行失败。该帮派区域未对玩家显示或未设置闪烁。
