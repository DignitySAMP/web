## 描述

获取指定服务器规则的标志位信息

## 返回值

返回服务器规则标志位

## 示例代码

```c
public OnGameModeInit()
{
    // 添加并配置Discord服务器规则
    AddServerRule("discord", "discord.gg/samp");
    SetServerRuleFlags("discord", 1);

    // 获取并打印规则标志位
    new E_SERVER_RULE_FLAGS:flags = GetServerRuleFlags("discord");
    printf("规则标志位: %d", _:flags); // 输出"规则标志位: 1"
    return 1;
}
```
