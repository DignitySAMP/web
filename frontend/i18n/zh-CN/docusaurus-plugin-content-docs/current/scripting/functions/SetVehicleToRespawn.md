## 描述

将车辆重置到其创建时的位置。

| 名称      | 说明            |
| --------- | --------------- |
| vehicleid | 要重置的车辆 ID |

## 返回值

**true** - 函数执行成功。

**false** - 函数执行失败。车辆不存在。

## 示例

```c
// 重置所有车辆
for (new i = 1; i < MAX_VEHICLES; i++)
{
    SetVehicleToRespawn(i);
}
```
