## 描述

强制终止指定玩家的物体编辑状态。

| 名称     | 描述                |
| -------- | ------------------- |
| playerid | 要终止编辑的玩家 ID |

## 返回值

该函数不返回特定值。

## 示例

```c
public OnPlayerCommandText(playerid, cmdtext[])
{
    if (!strcmp(cmdtext, "/stopedit", true))
    {
        EndObjectEditing(playerid);
        SendClientMessage(playerid, 0xFFFFFFFF, "服务器：物体编辑已终止！");
        return 1;
    }
    return 0;
}
```
