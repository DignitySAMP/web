## 描述

将车辆颜色索引转换为 HEX 格式颜色值（RGBA）。

## 参数

| 名称         | 描述                                        |
| ------------ | ------------------------------------------- |
| index        | [车辆颜色索引](../resources/vehiclecolorid) |
| alpha = 0xFF | Alpha 通道（默认值为 0xFF）                 |

## 示例

```c
new colour = VehicleColourIndexToColour(3, 0xFF); // 将索引3转换为不透明度FF的HEX颜色
```
