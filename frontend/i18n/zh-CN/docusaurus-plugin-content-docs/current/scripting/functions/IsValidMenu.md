## 描述

检测指定菜单 ID 是否有效。

| 参数名      | 说明            |
| ----------- | --------------- |
| Menu:menuid | 要检测的菜单 ID |

## 返回值

**true** - 菜单有效

**false** - 菜单无效

- [DisableMenuRow](DisableMenuRow): 全局禁用指定菜单行
- [AddMenuItem](AddMenuItem): 添加菜单选项
- [SetMenuColumnHeader](SetMenuColumnHeader): 设置菜单列标题
- [IsMenuDisabled](IsMenuDisabled): 检测菜单禁用状态
