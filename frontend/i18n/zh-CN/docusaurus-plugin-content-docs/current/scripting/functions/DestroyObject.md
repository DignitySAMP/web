## 描述

销毁通过 CreateObject 创建的物体。

| 名称     | 描述                                  |
| -------- | ------------------------------------- |
| objectid | 要销毁的物体 ID，由 CreateObject 返回 |

## 返回值

该函数不返回特定值。

## 示例

```c
public OnObjectMoved(objectid)
{
    DestroyObject(objectid);
    return 1;
}
```
