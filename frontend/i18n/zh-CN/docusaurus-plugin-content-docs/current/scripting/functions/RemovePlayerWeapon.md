## 描述

移除玩家指定的武器。

| 参数名          | 说明                                      |
| --------------- | ----------------------------------------- |
| playerid        | 需要移除武器的玩家 ID                     |
| WEAPON:weaponid | 要移除的[武器 ID](../resources/weaponids) |

## 返回值

**true** - 函数执行成功。

**false** - 函数执行失败。通常表示玩家未连接。

## 示例

```c
RemovePlayerWeapon(playerid, WEAPON_DEAGLE); // 移除玩家的沙漠之鹰手枪
```
