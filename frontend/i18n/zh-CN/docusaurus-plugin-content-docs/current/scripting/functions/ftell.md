## 描述

获取文件指针的当前位置。

| 名称        | 说明                                |
| ----------- | ----------------------------------- |
| File:handle | 要操作的文件句柄（由 fopen 返回）。 |

## 返回值

返回相对于文件起始位置的当前偏移量（整数值）。

## 示例

```c
// 以"只读"模式打开"file.txt"
new File:handle = fopen("file.txt", io_read);

// 检查文件是否成功打开
if (handle)
{
    // 操作成功

    printf("当前指针位置: %d", ftell(handle));

    // 关闭文件
    fclose(handle);
}
else
{
    // 操作失败
    print("文件\"file.txt\"不存在或无法访问。");
}
```
