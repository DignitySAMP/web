## 描述

检测指定玩家是否被允许使用武器。

| 参数名   | 说明              |
| -------- | ----------------- |
| playerid | 需要检测的玩家 ID |

## 返回值

**true** - 允许使用武器

**false** - 禁止使用武器

## 示例

```c
public OnPlayerSpawn(playerid)
{
    if (ArePlayerWeaponsAllowed(playerid))
    {
        // 执行武器相关操作
    }

    return 1;
}
```
