## 描述

获取拾取物的类型。

| 名称     | 描述                    |
| -------- | ----------------------- |
| pickupid | 需要获取类型的拾取物 ID |

## 返回值

返回拾取物的类型，  
当拾取物无效时返回 **-1**。

## 示例代码

```c
new g_Pickup;

public OnGameModeInit()
{
    g_Pickup = CreatePickup(1239, 1, 1686.6160, 1455.4277, 10.7705, -1);

    new type = GetPickupType(g_Pickup);
    // type = 1
    return 1;
}
```
