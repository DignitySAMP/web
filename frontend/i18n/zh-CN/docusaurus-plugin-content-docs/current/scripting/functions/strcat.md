## 描述

该函数用于将两个字符串拼接（连接）到目标字符串中。

| 参数                      | 说明                     |
| ------------------------- | ------------------------ |
| dest[]                    | 存储拼接结果的目标字符串 |
| const source[]            | 源字符串                 |
| maxlength = sizeof (dest) | 目标字符串的最大容量     |

## 返回值

新目标字符串的长度。

## 示例代码

```c
new string[40] = "Hello";
strcat(string, " World!");

// 字符串现在变为'Hello World!'
```
