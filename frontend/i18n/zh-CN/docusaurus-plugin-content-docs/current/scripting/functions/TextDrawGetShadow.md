## 描述

获取文本绘图文字的阴影尺寸

| 参数名      | 说明                          |
| ----------- | ----------------------------- |
| Text:textid | 需要获取阴影尺寸的文本绘图 ID |

## 返回值

返回文本绘图的阴影尺寸值（整数型）

## 示例

```c
new Text:gMyTextdraw;

public OnGameModeInit()
{
    gMyTextdraw = TextDrawCreate(100.0, 33.0, "示例文本绘图");
    TextDrawSetShadow(gMyTextdraw, 1);

    new shadow = TextDrawGetShadow(gMyTextdraw);
    // shadow = 1
    return 1;
}
```
