## 描述

获取文本绘图的字体颜色。

| 名称        | 说明                      |
| ----------- | ------------------------- |
| Text:textid | 需要获取颜色的文本绘图 ID |

## 返回值

返回文本绘图的字体颜色。

## 示例

```c
new Text:gMyTextdraw;

public OnGameModeInit()
{
    gMyTextdraw = TextDrawCreate(123.0, 123.0, "示例");
    TextDrawColor(gMyTextdraw, 0xFF0000FF);

    new color = TextDrawGetColor(gMyTextdraw);
    // color = 0xFF0000FF
    return 1;
}
```
