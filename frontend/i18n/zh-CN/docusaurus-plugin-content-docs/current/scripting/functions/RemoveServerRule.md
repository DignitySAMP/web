## 描述

移除指定的服务器规则。

| 参数名       | 说明                   |
| ------------ | ---------------------- |
| const rule[] | 要移除的服务器规则名称 |

## 返回值

成功执行返回 true，否则返回 false。

## 示例

```c
RemoveServerRule("rule");
```
