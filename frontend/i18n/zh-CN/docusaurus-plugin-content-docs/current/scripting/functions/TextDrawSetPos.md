## 描述

设置文本绘图在屏幕中的坐标位置（X 轴与 Y 轴）

| 参数名      | 说明                      |
| ----------- | ------------------------- |
| Text:textid | 需要设置位置的文本绘图 ID |
| Float:x     | 浮点型 X 坐标值           |
| Float:y     | 浮点型 Y 坐标值           |

## 示例

```c
new Text:gMyTextdraw;

public OnGameModeInit()
{
    gMyTextdraw = TextDrawCreate(100.0, 33.0, "示例文本绘图");

    TextDrawSetPos(gMyTextdraw, 200.0, 50.0);
    return 1;
}
```
