## 描述

重命名指定文件。

| 名称            | 说明                         |
| --------------- | ---------------------------- |
| const oldname[] | 原文件名（可包含完整路径）。 |
| const newname[] | 新文件名（可包含完整路径）。 |

## 返回值

**true** 操作成功，**false** 操作失败。

## 示例

```c
if (frename("example.txt", "file.txt"))
{
    // 操作成功
    printf("文件\"example.txt\"已重命名为\"file.txt\"。");
}
else
{
    // 操作失败
    print("文件\"example.txt\"不存在或无法访问。");
}
```
