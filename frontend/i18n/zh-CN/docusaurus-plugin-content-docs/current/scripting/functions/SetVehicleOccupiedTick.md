## 描述

设置车辆的占用时间戳。

## 参数

| 参数名    | 描述               |
| --------- | ------------------ |
| vehicleid | 要设置的车辆 ID    |
| ticks     | 要设置的时间戳数值 |

## 返回值

**true** - 函数执行成功

**false** - 函数执行失败

## 示例

```c
public OnGameModeInit()
{
    new vehicleid = CreateVehicle(560, 2096.1917, -1328.5150, 25.1059, 0.0000, 1, 8, 60);

    SetVehicleOccupiedTick(vehicleid, 300);
    return 1;
}
```
