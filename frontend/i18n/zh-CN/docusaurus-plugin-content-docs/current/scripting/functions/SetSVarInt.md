## 描述

设置整型服务器变量。

| 参数名       | 描述           |
| ------------ | -------------- |
| const svar[] | 服务器变量名称 |
| value        | 要设置的整数值 |

## 返回值

**true** - 函数执行成功

**false** - 函数执行失败（变量名为空或超过 40 字符）

## 示例

```c
// 设置版本号
SetSVarInt("Version", 37);

// 打印服务器当前版本
printf("版本号: %d", GetSVarInt("Version"));
```
