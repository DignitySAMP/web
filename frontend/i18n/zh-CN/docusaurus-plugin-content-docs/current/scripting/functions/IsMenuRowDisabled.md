## 描述

检查指定菜单行是否被禁用。

| 参数        | 说明              |
| ----------- | ----------------- |
| Menu:menuid | 需要检查的菜单 ID |

## 返回值

**true** - 菜单行已被禁用  
**false** - 菜单行未被禁用

## 相关函数

- [IsValidMenu](IsValidMenu): 验证菜单 ID 有效性
- [IsMenuDisabled](IsMenuDisabled): 检查菜单是否禁用
