## 描述

用于获取函数接收的参数数量

## 示例

```c
SomeFunction(...)
{
    printf("numargs(): %i", numargs());
}

public OnFilterScriptInit()
{
    SomeFunction(1, 2, 3);
}

// 输出: "numargs(): 3"
// 因为传递了三个参数（1, 2, 3）
```
