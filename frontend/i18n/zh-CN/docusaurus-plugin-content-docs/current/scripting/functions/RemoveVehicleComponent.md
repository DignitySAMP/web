## 描述

从车辆上移除指定组件。

| 参数名                                   | 说明            |
| ---------------------------------------- | --------------- |
| vehicleid                                | 目标车辆的 ID   |
| [component](../resources/carcomponentid) | 要移除的组件 ID |

## 返回值

**true** - 组件成功从车辆移除

**false** - 移除失败，通常表示车辆不存在

## 示例

```c
// 移除玩家车辆的氮气加速组件
new vehicleid = GetPlayerVehicleID(playerid);
RemoveVehicleComponent(vehicleid, 1010);
```
