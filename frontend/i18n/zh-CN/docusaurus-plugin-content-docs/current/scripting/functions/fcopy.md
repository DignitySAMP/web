## 描述

复制文件。

| 名称           | 描述                                       |
| -------------- | ------------------------------------------ |
| const source[] | 需要复制的（已存在的）文件名，可包含路径。 |
| const target[] | 新文件的名称，可包含完整路径。             |

## 返回值

**true** 表示成功，**false** 表示失败。

## 示例

```c
if (fcopy("example.txt", "file.txt"))
{
    // 成功
    printf("文件 \"example.txt\" 已成功复制到 \"file.txt\"。");
}
else
{
    // 错误
    print("文件 \"example.txt\" 不存在或无法打开。");
}
```

## 注意事项

:::warning

如果目标文件已存在，将被覆盖。

:::
