## 描述

设置浮点型服务器变量。

| 参数名       | 说明             |
| ------------ | ---------------- |
| const svar[] | 服务器变量名称   |
| Float:value  | 要设置的浮点数值 |

## 返回值

**true** - 函数执行成功

**false** - 函数执行失败（变量名为空或超过 40 字符）

## 示例代码

```c
// 设置版本号
SetSVarFloat("Version", 0.37);

// 打印服务器版本号
printf("当前版本: %.2f", GetSVarFloat("Version"));
```
