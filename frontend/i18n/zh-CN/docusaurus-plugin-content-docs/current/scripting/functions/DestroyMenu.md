## 描述

销毁指定的菜单。

| 参数名      | 说明            |
| ----------- | --------------- |
| Menu:menuid | 要销毁的菜单 ID |

## 返回值

**true** 销毁成功返回 true，否则返回 false

## 示例

```c
new Menu:exampleMenu;
exampleMenu = CreateMenu("Your Menu", 2, 200.0, 100.0, 150.0, 150.0);

// 后续操作...
DestroyMenu(exampleMenu);
```
