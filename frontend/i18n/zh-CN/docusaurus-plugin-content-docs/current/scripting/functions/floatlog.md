## 描述

该函数用于计算以指定底数的浮点数对数。

| 名称        | 描述                   |
| ----------- | ---------------------- |
| Float:value | 需要计算对数的浮点数值 |
| Float:base  | 对数的底数             |

## 返回值

计算结果（以浮点数形式返回）

## 示例

```c
public OnGameModeInit()
{
    printf("数值15.0以10.0为底的对数结果为 %.1f", floatlog(15.0, 10.0));
    return 1;
}
```
