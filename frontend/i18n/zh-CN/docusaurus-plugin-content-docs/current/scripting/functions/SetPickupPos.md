## 描述

设置拾取物的位置

| 参数名             | 说明                                          |
| ------------------ | --------------------------------------------- |
| pickupid           | 要设置的拾取物 ID                             |
| Float:x            | 拾取物新的 X 坐标                             |
| Float:y            | 拾取物新的 Y 坐标                             |
| Float:z            | 拾取物新的 Z 坐标                             |
| bool:update = true | 是否立即为所有玩家更新拾取物位置 (true/false) |

## 返回值

本函数始终返回 **true**

## 示例

```c
new g_Pickup;

public OnGameModeInit()
{
    g_Pickup = CreatePickup(1239, 1, 1686.6160, 1455.4277, 10.7705, -1);

    SetPickupPos(g_Pickup, 1958.5488, 1344.9137, 15.3613);
    return 1;
}
```
