## 描述

检查当前脚本中是否存在特定公共变量。

| 名称         | 描述                 |
| ------------ | -------------------- |
| const name[] | 要检查的公共变量名称 |

## 返回值

**true** - 变量存在

**false** - 变量不存在

## 相关函数

- [getpubvar](getpubvar): 获取公共变量值
- [setpubvar](setpubvar): 设置公共变量值
- [numpubvars](numpubvars): 统计脚本中公共变量数量
