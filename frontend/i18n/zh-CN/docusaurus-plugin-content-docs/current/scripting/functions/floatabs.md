## 描述

返回指定浮点数的绝对值。

| 名称        | 描述                   |
| ----------- | ---------------------- |
| Float:value | 要计算绝对值的浮点数值 |

## 返回值

浮点数的绝对值（以浮点数形式返回）

## 示例

```c
new Float:value;

value = floatabs(47.0); // 返回47.0

value = floatabs(-47.0); // 同样返回47.0
```
