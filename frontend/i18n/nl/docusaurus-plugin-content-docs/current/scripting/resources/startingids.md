| File | 0 |
| GangZone | 0 |

| Pickup | 0 |
| Player | 0 |
| Player Class | 0 |
| TextDraw / PlayerTextDraw | 0 |
| Timer | 1 |

Let op: objecten en voertuigen starten bij 1; trek 1 af bij array‑indexering.
