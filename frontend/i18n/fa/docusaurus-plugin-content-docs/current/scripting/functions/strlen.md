## توضیحات

طول یک رشته رو می‌گیره.

| اسم           | توضیح                      |
| -------------- | -------------------------------- |
| const string[] | رشته‌ای که می‌خوای طولش رو بگیری. |

## مقادیر برگشتی

طول رشته به شکل عدد صحیح.

## مثال‌ها

</div>

```c
new stringLength = strlen("This is an example string."); // stringLength حالا 26 میشه
```
