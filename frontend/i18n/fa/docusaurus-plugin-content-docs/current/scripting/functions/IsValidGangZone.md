## توضیحات

چک کردن اینکه gangzone معتبره یا نه.

| نام     | توضیحات               |
| ------- | --------------------- |
| zoneid  | ID gangzone.          |

## مقادیر برگشتی

**true** - gangzone معتبره.

**false** - gangzone معتبر نیست.
