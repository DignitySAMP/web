## توضیحات

چک می‌کنه که آیا ID یه custom model معتبره یا نه.

| نام     | توضیحات                             |
| ------- | ----------------------------------- |
| modelid | ID custom model که باید چک بشه.     |

## مقادیر برگشتی

**true** - ID custom model معتبره.

**false** - ID custom model معتبر نیست.
