## توضیحات

این کالبک زمانی فراخوانده می‌شود که NPC از وسیله نقلیه خارج شود.

## مثال‌ها

</div>

```c
public OnNPCExitVehicle()
{
    print("The NPC left the vehicle");
    return 1;
}
```
